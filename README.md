# 8kno Homebrew Tap

This repository is generated from the private 8kno source tree.

## Install

```bash
brew tap 8kno/tap
brew install 8kno
```

## Contents

- The tap exposes the `8kno` install name after `brew tap 8kno/tap`.
- Runtime archives are published as GitHub Release assets for this repository.
- The formula points to `https://github.com/8kno/homebrew-tap/releases/download/v0.1.1`.
